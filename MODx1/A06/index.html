Conhecendo novas tags para texto!

A tag <sup>:

A tag sup é usada para reprensetar um texto elevado,
ou seja, um texto menor acima do outro, muito utilizado em
operações matemáticas. Mas também utilizado para abreviações.

EXEMPLO:

23 + 40 (2*4)3
Sra | Sro

A tag <sub>:

A tag sub tem o mesmo sentido porém muito utilizada em
operações químicas.

EXEMPLO:

H2O | CH 4

A tag <big>:

A tag big é uma tag já defasada que caiu em desuso, isso porquê
ela faz referencia ao formato do texto, e não faz sentido para
o texto ter uma palavra ou sentença maior, precisamos dar sentido
as coisas.

A tag <ins>:

A tag ins, é para demonstrar que uma palavra foi inserida dentro do texto.

EXEMPLO

Essa palavra que vou digitar agora, Foi inserida no texto.

A tag <mark>:

A tag mark, é para demonstrar uma palavra que foi marcada.

EXEMPLO

Essa palavra que vou digitar agora, está extremamente marcada.
